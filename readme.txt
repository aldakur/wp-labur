== Description ==

labur is a quick, modern, and open-source link shortener for basque community. This plugin allows you to use labur service in Wordpress.
labur software is a good way to shorten the long urls. Now you can use it very fast inside your Wordpress for your posts.

= Plugin Features =
* Shorten url of posts.
* Github: [labur Wordpress Plugin](https://github.com/aldakur/wp-labur)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-labur` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->labur screen to configure the plugin
